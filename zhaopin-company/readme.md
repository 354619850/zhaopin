1.“首页”模块：
（1）人才搜索功能：企业用户可根据需要搜索人才库；
（2）搜索历史列表：显示用户最近的搜索历史；
（3）热搜推荐列表：将被高频搜索的求职信息推荐给企业；
2.“候选人”模块：
（1）搜索功能：从候选人列表中搜索求职信息；
（2）候选人列表：显示已发布职位的候选人列表，可设置收藏和自动过滤；
3.“职位”模块：
（1）发布职位功能：企业可发布职位信息；
（2）编辑职位功能：企业可修改已发布的职位信息；
（3）已停止职位功能：企业可停止已发布的职位(停止后该职位信息对应聘者不可见)
4.“消息”模块：
（1）反馈消息列表；显示反馈消息
5.“我”模块：
（1）公司档案功能：企业用户可编辑公司信息
（2）已收藏人才：可查看自己收藏的人才求职信息
（3）设置：设置公司信息是否公开，是否接收简历投递等
